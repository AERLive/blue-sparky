Get Started

Get Started

The purpose of Sparky is to help you...

  • Use Bower to Grab the latest code from Bootstrap 3 and its dependencies. This helps you avoid missing any updates they make.
  • Use grunt to reference the Bootstrap Sass files and update variables
  • Use grunt and postcss to make sure your CSS accounts for vendor prefixes
  • Add new javascript plugins and custom scripts and compress them into one file to decrease page load
  • Save time starting a new project

Step 1

Make sure you have the following installed


Install Node.js

Install bower
npm install -g bower

Install libSass
npm install -g grunt-cli

Step 2

Fork this repo to your account

Step 3

Install node packages - looks at package.json
npm install

Step 4

Install bower packages - looks at bower.json
bower install

Step 5

start grunt watch
Update sass files
